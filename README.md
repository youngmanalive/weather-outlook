# Weather Outlook

A small static React app that turns hourly weather data into a practical daily outlook.

Most weather apps overstate low-impact rain by leading with a single icon. This app focuses on whether the forecast should actually affect plans: timing, amount, duration, temperature, wind, and humidity.

## Features

- Today, tomorrow, and 3-day outlooks
- Location search with Open-Meteo geocoding
- Browser location with reverse-geocoded display name
- Local location persistence
- Practical rain impact language
- Static GitHub Pages deployment

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
